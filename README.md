# get-fan-mode-asus
script to get the fans mode in asus laptops


##how to use

```bash
┌──(test㉿test)-[~]
└─$ get-fan-mode-asus 
default mode
```

              
```bash
┌──(test㉿test)-[~]
└─$ get-fan-mode-asus 
overboost mode
```



```bash
┌──(test㉿test)-[~]
└─$ get-fan-mode-asus 
silent mode
```
